# LOGGING DOCS


## Intro
*logging* - это библиотека логгирования для C++, поддерживающая безопасный вывод логов в однопоточных и многопоточных приложениях


## Quick start

Для того, чтобы воспользоваться логгером, вам необходимо определить его статические поля, которые отвечают за глобальные настройки. Сделать это вы можете с помощью метода *<logging/logger.hpp>* ```logger::Logger::SetupLogger```

#### Параметры:

 - **file_path** - Путь к файлу для записи логов. Если он не задан, то вывод будет осуществляться в стандартный поток
 - **logging_level** - Настройка уровня вывода логов. ```debug -> info -> warning -> exception -> error -> critical```
 - **io_mode** - Режим вывода логов
    - *Синхронный режим:* обычный режим вывода при работе только с 1 потоком
    - *Асинхронный режим:* специальный режим для одновременного обслуживания нескольких потоков. Писатель и логгер взаимодействуют через очередь. При попытке взаимодействия с ней потоки синхронизируются с помощью *mutex*
 - **colored** - Включение режима цветного вывода логов

После настройки логгера вы можете в любом месте кода создать экземпляр его класса, передав в конструктор желаемое имя, и взаимодействовать с ним через его методы для непосредственного вывода логов
